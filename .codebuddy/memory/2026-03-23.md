## Bugfix: 识字学习建议页面顶部白色背景

- **问题**：`literacy-advice.vue` 在微信小程序中顶部显示白色背景条
- **根因**：返回按钮使用 `position: fixed` 脱离文档流，导致 `.main-card`（有 `margin-top: 246rpx`）成为第一个占据空间的元素，页面顶部区域无实际内容填充，`page` 节点默认白色背景暴露
- **修复**：参考 `science-principle.vue` 的实现方式：
  - 返回按钮改为文档流定位（`display: inline-flex`，去掉 `position: fixed`）
  - 添加 `.safe-area-top`（`height: 88rpx`）和 `.safe-area-bottom`（`calc(48rpx + env(safe-area-inset-bottom))`）
  - `.main-card` 的 `margin-top` 从 `246rpx` 改为 `32rpx`
  - 移除 `.page-container` 的 `position: relative`
