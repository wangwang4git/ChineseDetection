# 变更提案：个人页增加用户年龄功能

## 状态
Complete

## 概述

为个人页用户信息卡片增加年龄展示和编辑功能，帮助家长记录孩子的年龄信息，为后续识字量评估提供参考依据。

## 背景

当前个人页用户信息卡片展示头像、昵称和账号（OpenID），缺少年龄信息。根据 Figma 设计稿，需要在账号下方增加年龄展示行，格式为"年龄：X岁"。

## 目标

1. 在个人页用户信息卡片的账号下方增加年龄展示
2. 扩展用户数据对象，增加 `age` 字段
3. 更新首次访问引导弹窗，提示用户输入年龄
4. 点击年龄字段弹出数字选择器，支持用户输入年龄
5. 年龄数据仅保存在本地，不同步到后台

## 变更范围

### 文件变更

| 文件 | 变更类型 | 说明 |
|------|---------|------|
| `src/pages/profile/profile.vue` | 修改 | 增加年龄展示、点击事件、数字选择弹窗 |
| `src/utils/userManager.js` | 修改 | 扩展 UserInfo 类型，增加 age 字段和更新方法 |

### 功能变更

| 功能 | 变更说明 |
|------|---------|
| 用户信息展示 | 账号下方增加年龄行 |
| 用户信息编辑 | 点击年龄弹出数字选择器 |
| 引导弹窗 | 增加年龄输入提示 |
| 数据模型 | UserInfo 增加 age 字段 |

## 非目标

- 年龄数据不同步到后台云数据库
- 不添加年龄相关的统计分析功能
- 不修改其他页面

## 验收标准

1. 个人页用户信息卡片在账号下方显示"年龄：X岁"
2. 点击年龄行弹出数字选择器（1-15岁范围）
3. 选择年龄后立即更新UI并保存到本地存储
4. 首次访问引导弹窗包含年龄输入提示
5. 微信小程序和 H5 环境功能一致
