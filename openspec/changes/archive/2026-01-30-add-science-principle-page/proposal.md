# Proposal: 新增识字量测试科学原理页

## 背景

当前首页副标题仅显示"一起来测测认识多少字吧！"，用户无法了解检测背后的科学原理和测试策略。根据设计稿要求，需要：

1. 首页副标题区域增加"科学原理查看请参考👉"链接
2. 新增"识字量测试科学原理"页面，详细介绍测试方法

## 变更范围

### 1. 首页修改 (home.vue)

- 修改副标题文案，增加"科学原理查看请参考👉"可点击链接
- 增加跳转到科学原理页的导航功能

### 2. 新增科学原理页 (science-principle.vue)

- 页面路径：`pages/science-principle/science-principle`
- UI 完全还原 Figma 设计稿 (272_308)
- 内容包括：
  - 数据来源说明（2500 常用字，覆盖 98.5% 使用频率）
  - 测试策略介绍（分层频率抽样）
  - 识字量计算公式展示
  - 常用字层级划分表格（L1-L6）
  - 测试结束机制（熔断机制）
  - 计算示例演示

### 3. 路由配置 (pages.json)

- 新增科学原理页路由

### 4. 分享功能配置 (share.js)

- 新增 `getSciencePrincipleShareConfig()` 分享给好友配置函数
- 新增 `getSciencePrincipleTimelineConfig()` 分享到朋友圈配置函数
- 分享标题：🔬 识字量测试科学原理 - 了解科学测评方法
- 分享路径：跳转到科学原理页

## 影响分析

| 影响项 | 说明 |
|--------|------|
| 现有功能 | 不影响现有功能，仅新增页面和入口 |
| 代码改动 | 首页小改动 + 新增页面 |
| 依赖变更 | 无 |
| 数据存储 | 无 |
| 分享配置 | share.js 新增 2 个配置函数 |

## 验收标准

1. 首页副标题显示两行：
   - 第一行：一起来测测认识多少字吧！（紫色 #6E11B0）
   - 第二行：科学原理查看请参考👉（蓝色 #155DFC，可点击）
2. 点击"科学原理查看请参考👉"跳转到科学原理页
3. 科学原理页 UI 与设计稿一致
4. 返回按钮可返回首页
5. 支持微信分享功能
