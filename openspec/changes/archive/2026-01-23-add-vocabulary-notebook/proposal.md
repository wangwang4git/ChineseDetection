# 生字本功能

## 动机

用户在完成汉字检测后，需要一个集中管理"不认识汉字"的地方，方便后续学习和复习。当前系统虽然在历史记录中保存了 `unknownChars`（需加强汉字），但缺乏独立的学习入口和互动式复习功能。

## 目标

1. 在个人页增加"生字本"入口，展示待学习汉字数量
2. 创建生字本页面，展示所有需加强的汉字（去重合并）
3. 支持点击汉字进入学习模式，复用测试页 UI，实现认识/不认识互动
4. 认识后从生字本移除，不认识则保留

## 范围

- **新增**: 生字本页面 (`pages/vocabulary-notebook/vocabulary-notebook.vue`)
- **修改**: 个人页 (`pages/profile/profile.vue`) - 添加生字本入口
- **修改**: 测试页 (`pages/test/test.vue`) - 支持生字学习模式（复用）
- **修改**: 本地存储 (`utils/storage.js`) - 生字本数据管理

## 非目标

- 不支持云端同步生字本（仅本地存储）
- 不支持手动添加生字（仅通过检测结果自动添加）
- 不支持生字分类/标签功能
