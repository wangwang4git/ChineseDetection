## 组件更新

### RiceGrid.vue 更新
- 背景改为白色
- 边框改为金色 `#FDC700`
- 移除米字格内部红色线条，改为更简洁的设计

### CharacterCard.vue 更新
- 新增 `variant` prop 支持不同样式变体
- 默认样式改为粉色渐变背景
- 调整尺寸为 104x150rpx
- 调整圆角为 20rpx

## 兼容性考虑

### 微信小程序兼容
- 渐变背景使用 `linear-gradient` 兼容性良好
- 阴影使用 `box-shadow` 需注意部分旧版本兼容
- 使用 `#ifdef MP-WEIXIN` 处理平台差异

### 响应式适配
- 所有尺寸使用 `rpx` 单位
- 按钮宽度使用 `flex: 1` 自适应
- 卡片宽度使用百分比或固定 rpx
